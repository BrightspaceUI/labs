ComputedValues Controller

This demo component simply shows how the controller may be used. To get a better idea of what functionality and options this controller supports, take a look at the demo-component source code.
  • The fullName computed value is dependant on the firstName and lastName
  • The screamingFullName computed value is dependant on the fullName computed value
  • The asyncShortName computed value shows how a computed value can be async and that it can have custom shouldCompute logic
  • The altFullName computed value shows that you can use a single computed value instead of a collection of them
